- [Round Trip Time/Ping](https://youtu.be/PhjHXeMNpp8?si=HFhgZPBjIxVIjMZm&t=6757)
  - Latency calacualtes only one way delay, but RTT is a time required to send data from one device to another and back
  - It matters in real time apps, where going forth of data is so common like video calling apps and real time streaming apps and basically games
  - RTT for video / audio calls below than 300 ms is unnoticeable

- [Ping utility use for measuring ping between two devices( like host and the server )](https://youtu.be/PhjHXeMNpp8?si=QMYyogK3TZDRmCbR&t=6927)
  - ex: ping 8.8.8.8 ( google DNS server IP ) measures the ping timing b/w our host and google server

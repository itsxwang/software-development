[Layers of OSI Model](https://youtu.be/hWIktHvNjeM?si=7CNtO0oFep2XvdY-&t=207):
1. Physical Layer - Responsible for the physical connection between devices, including cables, switches, and hubs
2. Data Link Layer - Responsible for the logical connection between devices, including MAC addresses and error detection
3. Network Layer - Responsible for the logical addressing and routing of packets, including IP addresses and routing protocols. In this layer, routers what path is short, reliable, and less congested to send the data to the destination.
4. [Transport Layer](https://youtu.be/hWIktHvNjeM?si=jThUfEMw0KwmIacc&t=437) - Responsible for the end-to-end communication between applications, including TCP and UDP protocols
5. Session Layer - Responsible for establishing, managing, and terminating sessions between applications. And Divides data into chunks called `packets`.  
6. Presentation Layer - Responsible for data formatting, encryption, and compression
7. Application Layer - Responsible for providing network services to end-user applications
